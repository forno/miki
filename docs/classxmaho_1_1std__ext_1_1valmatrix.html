Detailed Description

template<typename T>
class xmaho::std_ext::valmatrix< T >

The adapter class for matrix valarray.

valarray class are useful for multiple dimension vector. But it haven't dimension values. This class hold dimension value and present some API.

Invariant
size() == row_size() * col_size()
Template Parameters
TValue type of valarray.
